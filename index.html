Project Exam 1

Project exam 1 - Spring 2022

Tools/technology used:

    • Planning: Trello
    • Design: Figma & Pixelmator
    • Implementation: HTML, CSS & JavaScript
    • Version Control: Github
    • Host: Netlify

About the project

The exam task consisted of creating my own blog, with a topic og my choice. Here I was going to use Wordpress as a Headless CMS, and design my own front end. The requirement was to create a full-fledged website, which included a slideshow, loadmore function for the posts and a image modal.

How it went

I set up the page statically and added all the CSS before starting the technical stuff (JS). Added a function to create image modal for each post. On the posts page i extracted only 9 posts to the first page, and constructed a button that ectracted the last 3.
